# VowSuite - White-Label Wedding Planning Platform

**The Future of Weddings is Digital.**

VowSuite is a premium, white-label SaaS solution designed for wedding planners to scale their business. It replaces scattered emails and spreadsheets with a centralised digital hub, offering a personalised app experience for every couple and a powerful management dashboard for the planner.

##  Key Features

###  Planner Dashboard
The command center for the wedding planner.
-   **Client Management**: Overview of all active weddings with status indicators.
-   **Global Checklist**: Track progress across multiple clients in one view.
-   **Smart Calendar**: A realistic "Busy Planner" schedule that manages client meetings, vendor appointments, and personal life logic.
-   **Settings**: Personalized profile and notification preferences.

###  Bride App (Client View)
A personalised planning tool for the couple.
-   **Planning Dashboard**: Dedicated view for the bride to manage her big day.
-   **Guest Manager**: Add guests, assign tables, and track RSVPs.
-   **Budget Tracker**: Monitor deposits, remaining balances, and vendor payment status.
-   **Checklist**: A personalised to-do list for the couple.
-   **QR Code Sharing**: Instantly share the Guest App with friends and family via a generated QR code.

###  Guest App (Mobile View)
A mobile-first experience for wedding guests.
-   **Mobile Toggle**: Simulate the mobile experience on desktop with a single click.
-   **Smart Video Hiding**: Intro videos automatically hide on mobile devices to focus on essential content.
-   **Event Schedule**: Interactive timeline of the wedding day.
-   **Digital Menu**: View menu options and dietary information.
-   **Seating Plan**: Searchable seating chart to find tables instantly.
-   **Photo Sharing**: Guests can upload photos directly to the couple's digital album.

##  Tech Stack
-   **Framework**: React (Vite)
-   **Styling**: Tailwind CSS
-   **Icons**: Lucide React
-   **Routing**: React Router DOM
-   **Language**: TypeScript

##  Installation & Setup

1.  **Navigate to the app directory**:
    ```bash
    cd vowsuite/app
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

##  Mobile First Design
VowSuite is built with a "Mobile First" philosophy. The Guest App is optimized for handheld devices, ensuring a seamless experience for guests attending the wedding. The "Mobile View Toggle" allows planners to demonstrate this capability to clients directly from their desktop.
